# Story 3.5: Course & Lesson Card Components

## Status
Draft

## Story
**As a** developer,
**I want** reusable course and lesson card components,
**so that** consistent UI is maintained across dashboard, course list, and lesson list.

## Acceptance Criteria
1. `CourseCard` component created with props: course, progress, showContinue
2. `LessonCard` component created with props: lesson, isCompleted, isLocked
3. `ProgressBar` component created with props: progress (0-1), label
4. Components use consistent Tailwind styling
5. Cards are clickable with hover states
6. Cards responsive on mobile (full width on small screens)
7. Components have TypeScript prop interfaces exported
